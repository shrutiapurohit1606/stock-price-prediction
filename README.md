# Stock Price Prediction
A Python machine learning project that predicts future stock prices using historical data from Yahoo Finance.
## Features
* Downloads stock data automatically
* Trains a prediction model
* Evaluates model using Mean Squared Error (MSE)
* Visualizes stock trends and predictions
* Predicts the next 30 days of prices
## Technologies
* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* yFinance
## Output
* Stock price graph
* Predicted price graph
* Future price predictions
## Run the Project

Install required libraries:

```
pip install pandas numpy matplotlib scikit-learn yfinance

Run the script:

```
python stock_prediction.py
```
